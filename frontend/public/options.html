Microphone setup

This extension needs microphone access for voice answers. Chrome will ask for permission the first time this page requests audio.

If you denied access earlier, use the site controls in the address bar on this page and switch Microphone to Allow, then try again.

Requesting microphone access...
  1. Click Allow Microphone.
  2. Approve the browser prompt if it appears.
  3. Return to the extension popup and try the mic button again.
